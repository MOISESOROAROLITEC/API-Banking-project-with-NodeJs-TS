# How to run this app

## **Install dependances**
>
>```bash
> npm install
>```

## **Running tests**
>
>```bash
> npm run test
>```

## **Running server**

### Générer le model prisma
>
>```bash
> prisma generate
>```

### Transcode en js
>
>```bash
> npm run compile
>```

### Start server
>
>```bash
> npm run start
>```
